START HERE » Intel® XDK IoT Edition - Standalone Installation » Install Bonjour

  1. Download Bonjour via Bonjour Print Services for Windows.

    1. Go to http://support.apple.com/kb/DL999.

    2. Click “Download” to download BonjourPSSetup.exe to your computer.

  2. Double-click on the downloaded BonjourPSSetup.exe to start the installer.

  3. Follow the instructions in the installation wizard.
