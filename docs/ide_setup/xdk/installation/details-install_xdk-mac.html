START HERE » Intel® XDK IoT Edition - Standalone Installation » Install Intel® XDK - Mac

  1. Get the latest Intel® XDK IoT Edition installer.

    1. On the USB key: downloads → Mac → edison_media.
    2. Copy the iot_web_mac_master_[version].dmg installer file to your computer.

  2. Double-click on iot_web_mac_master_[version].dmg to open the Apple Disk Image.

  3. Double-click on the extracted xdk_full_[version].pkg to start the installer.

  4. After launching the Intel® XDK installer, follow the installation wizard and click Next where needed.
