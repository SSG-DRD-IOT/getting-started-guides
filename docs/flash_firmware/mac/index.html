START HERE » Flash Firmware using Flash Tool Lite » Set up Flash Tool Lite for Mac

  1. Get the latest Flash Tool Lite.

    1. On the USB key: downloads → Mac
    2. Copy PhoneFlashToolLite_[version]_mac64.dmg to your computer.

  2. Double-click on PhoneFlashToolLite_[version]_mac64.pkg to start the installer.

  3. Follow the installation wizard prompts. Click “Next” where needed.
