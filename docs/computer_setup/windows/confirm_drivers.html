Open Windows Device Manager

Launch the Device Manager utility on Windows in order to debug your Intel® IoT board USB connections. View detailed instructions »

Confirm installation of Intel® Edison Drivers

Look for both “Intel Edison USB Composite Device” and “Intel Edison Virtual Com Port” items in Device Manager under “Ports (COM & LPT)” after plugging in the device mode USB cable. View detailed instructions »

Confirm installation of FTDI serial drivers

Look for “USB Serial Port” in Device Manager under “Ports (COM & LPT)” after plugging in the UART/serial USB cable. View detailed instructions »
